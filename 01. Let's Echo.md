#  Let's Echo
Write a bash script which does just one thing: saying "HELLO".

## Input Format

There is no input file required for this problem.

## Output Format

    HELLO

## Sample Input

    -

## Sample Output

    HELLO

## Script

    echo "HELLO"

* **echo** - is the command to print
* **"HELLO"** - can be any unique string which is to be printed
